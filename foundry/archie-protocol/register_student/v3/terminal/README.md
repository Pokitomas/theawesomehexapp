## Architecture actually evaluated

The learned component is the immutable quantized v2 route expert: a linear classifier over word, bigram, trigram, character 3–5 gram, and structural features. V3 leaves all learned weights unchanged and repairs two generic compositional boundaries in the controller:

1. `instead of` no longer triggers correction replacement.
2. `and afterward`, `following completion`, `upon completion`, and `followed by` are recognized as ordered clause boundaries.

Deterministic code remains limited to fail-closed authority, context/support sufficiency, reference resolution, raw-source abstention, clause activity, and ordered composition. The admitted production register projection was not edited.
