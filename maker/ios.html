MAKER / IPHONE + IPAD

Open Maker like an app.

No App Store account or setup screen. Open Maker, then add it to your Home Screen for a full-screen app icon.

OPEN MAKER

Install on iPhone or iPad

  1. Open this page in Safari.
  2. Tap the Share button ⇧.
  3. Choose Add to Home Screen.
  4. Tap Add. Maker will appear with your other apps.

Maker stores drafts on this device and reads public GitHub status. Repository writes still require the authorized GitHub workflow; the browser never receives repository or model credentials.
