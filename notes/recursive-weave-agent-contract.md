# Recursive Weave Agent Contract

Each implementation lane must publish:

- a named remote branch and draft PR;
- exact owned files and dependencies;
- typed input and output events;
- deterministic replay witness;
- hostile duplicate, contradiction, stale-input, and budget-exhaustion witnesses;
- a concise rationale receipt citing source event IDs;
- no private chain-of-thought or hidden model scratchpad;
- no merge outside the final assembly vehicle.

Every model adapter is replaceable. The shared belief/plan graph is canonical. A model response that cannot be represented as typed events is advisory residue, not state.
