# Recursive Weave Sprawl

The next weave generation is a collective cognition substrate, not a second model and not a prose chatroom.

It must turn many agent outputs into one inspectable evolving state:

1. typed beliefs, questions, contradictions, evidence, goals, plans, decisions, tests, and artifacts;
2. a deterministic fold that preserves provenance and disagreement;
3. a bounded deliberation planner that derives the next useful roles and prompts from unresolved state;
4. replaceable model adapters that execute those roles without owning canonical state;
5. recursive waves whose outputs become the next wave's inputs;
6. convergence receipts proving why a belief, plan, or artifact advanced;
7. explicit uncertainty, dissent, minority reports, and rollback residue;
8. hard recursion, cost, privacy, replay, and mutation boundaries.

The canonical mind is the event-sourced belief/plan graph plus its derivation receipts. Agents are transient inference organs. GitHub and the Universal Remote remain the authority skeleton.

## Non-negotiable boundaries

- No hidden chain-of-thought persistence. Store claims, evidence, decisions, uncertainty, and concise rationale only.
- No model output directly mutates canonical state; every change is a typed event with provenance.
- No automatic recursion without a finite budget, deduplication key, stop condition, and human-visible trace.
- No consensus-by-majority. Contradictions remain first-class until evidence or an explicit decision resolves them.
- No global administrator abstraction.
- No secrets or private payloads in public projections.
- No self-triggering workflow may execute untrusted branch code with write credentials.
- Every recursive wave must be replayable from the immutable event stream.

## Target loop

`observe -> retrieve -> propose -> oppose -> test -> synthesize -> decide -> plan -> dispatch -> observe`

The loop advances only through durable receipts. Silence is not progress. Fluency is not evidence. A synthesis that cannot cite its inputs is invalid.
